It looks like there is a difference between the length of Vermilion Rockfish in the MPA and the REF site is it statistically different? Lets find out.

T-Test Hypothesis Testing

Null hypothesis or H0 : There is NO difference in length of Vermilion Rockfish between the MPA and the REF site. Another way to say this: the true difference in means between MPA and REF is 0.

Alternative hypothesis or Ha: There is a difference in length of Vermilion Rockfish between the MPA and the REF site. Another way to say this: the true difference in means between MPA and REF is not 0.

REMEMBER a small p-value will indicate more evidence in favor of the alternative hypothesis (we reject the null hypothesis), which means there is a difference, while a large p-value will indicate that we dont have enough evidence to reject the null hypothesis, which means there is no statistical difference that we can prove with our data.

t_test_length <- t.test(length_cm ~ site, data = data_rq_1)

t_test_length

In general, a one-way ANOVA is considered to be fairly robust against violations of the normality [and equal variance] assumption as long as the sample sizes are sufficiently large.

A little messier, but it looks like there are some differences visually AT LEAST between some groups

ANOVA Hypothesis Testing

Null hypothesis or H0 : There are NO difference in length of Vermilion Rockfish between any of the areas. Another way to say this: the true difference in means between all the areas is 0.

Alternative hypothesis or Ha: There is a difference in length of Vermilion Rockfish between the MPA and the REF site. Another way to say this: the true difference in means between MPA and REF is not 0.

anova_model <- aov(length_cm ~ area, data = data_rq_2)
